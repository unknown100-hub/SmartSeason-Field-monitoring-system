## Project Overview

SmartSeason is a comprehensive agricultural field monitoring dashboard that I developed to help farm managers and agronomists track, manage, and monitor their fields effectively. The system provides real-time insights into crop health, growth stages, and field status based on key agricultural metrics.

---

## Key Achievements

### 1. Complete Full-Stack Implementation
- **Frontend**: Built with React 19 and Vite for fast, responsive user experience
- **Backend**: RESTful API using Express.js with proper error handling
- **Database**: MySQL with optimized queries and auto-table creation

### 2. Smart Field Status Computation
The system automatically calculates field status based on:
- **Moisture levels** - Fields below 45% moisture are flagged as "At Risk"
- **NDVI values** - Normalized Difference Vegetation Index for crop health
- **Completion percentage** - Tracks growth progress toward harvest
- **Stage tracking** - Planted → Growing → Ready → Harvested

### 3. Role-Based Access Control
- Regional agronomists have full dashboard access
- Field agents can view and update their assigned fields
- Managers have view-only access

### 4. Resilient Design
- Demo mode with localStorage fallback when backend is offline
- Automatic health check and reconnection
- Graceful degradation for better user experience
